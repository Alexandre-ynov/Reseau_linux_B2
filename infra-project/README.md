# Infrastructure Project

Ce projet contient la configuration complète de l'infrastructure réseau Linux pour l'entreprise cliente.

## Architecture

Voir le schéma d'infrastructure : `schema-infra.png`

## Composants

### Firewall
Configuration du pare-feu avec nftables pour sécuriser le réseau.

### VPN WireGuard
Configuration du VPN WireGuard pour les accès sécurisés distants.

### Bastion
Configuration du serveur bastion pour l'administration sécurisée.

### Web Server (Nginx)
Configuration du serveur web Nginx.

### Database (MariaDB)
Configuration de la base de données MariaDB.

### Observability
Outils de monitoring et observabilité de l'infrastructure.

### Backups
Stratégies et configurations de sauvegarde.

### Security
Politiques et configurations de sécurité supplémentaires.
